# First Login, Initial Setup, and License Activation

After deployment, open:

- `https://n8n.yourname.synology.me`

## 1) Create owner account

On first screen:

- Enter your own admin credentials
- Continue onboarding

## 2) Workspace onboarding

- Fill company/workspace information
- Complete initial setup prompts

## 3) Activate n8n license (if prompted)

In n8n UI:

- Open `Settings`
- Navigate to `Usage and plan`
- Enter activation key
- Click `Activate`

## 4) Optional personalization

- Go to `Personal`
- Change theme (for example, Dark)
- Save

## 5) Quick post-install checks

- Create a test workflow
- Add a webhook trigger and test URL generation
- Confirm executions are persisted after container restart
